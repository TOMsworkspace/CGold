Ubuntu GCC
==========

The ``GCC`` compiler is usually used on ``Linux OS``. To install it on ``Ubuntu`` run:

.. code-block:: none

  > sudo apt-get install -y gcc

Check the location and version

.. code-block:: none

  > which gcc
  /usr/bin/gcc

  > gcc --version
  gcc (Ubuntu 4.8.4-2ubuntu1~14.04.3) 4.8.4
